Esses são os nossos módulos de Web Designer

O curso de Webdesigner é um pacote composto por vários cursos ligados à área de web designer. O aluno irá aprender a criar sites para serem publicados na Internet.

  • Presencial e Online
  • 4 módulos
  • Carga horária média: 180 horas

HTML

Saber mais

Carga horária: 30 horas

Pré-requisitos: O aluno precisa conhecer o Windows e ter feito o curso de Internet.

O objetivo do curso é ensinar o aluno os comandos da programação HTML de um jeito bem fácil e interessante, aprendendo a linguagem enquanto monta três sites Web diferentes, aplicando os comandos que forem aprendidos.

FireWorks

Saber mais

Carga horária: 30 horas

Pré-requisitos: O aluno precisa conhecer o Windows e saber navegar na Internet. É também recomendável que tenha noções da linguagem de programação para a Web, HTML.

O objetivo do curso é preparar o aluno para utilizar a maioria dos recursos disponíveis no Fireworks, habilitando-o a criar páginas e sites para a Internet, de forma fácil e eficiente, sem programação.
